\medskip
\textbf{1. Description of $L(A^{(1)})$ and $L(A^{(2)})$.}

\medskip
\underline{$A^{(1)}$:}

\begin{center}
\begin{tikzpicture}[shorten >=1pt,node distance=2.5cm,on grid,auto]
\node[state,initial] (1) {$1$};
\node[state,accepting] (2) [right=of 1] {$2$};
\node[state] (3) [right=of 2] {$3$};
\node[state,accepting] (4) [below=of 2] {$4$};

\path[->]
(1) edge node {$a$} (2)
    edge node[left] {$b$} (4)
(2) edge node {$a$} (3)
    edge node[right] {$b$} (4)
(3) edge[loop above] node {$a,b$} ()
(4) edge[bend left=15] node {$a$} (2)
    edge node[below left] {$b$} (3);
\end{tikzpicture}
\end{center}

The accepting states are $\{2,4\}$ and state $3$ is a non-accepting sink (it loops on both $a$ and $b$).
From state $2$, reading $a$ goes to the sink; from state $4$, reading $b$ goes to the sink.
Thus, once the first symbol is read, the letters must alternate in order to avoid the sink.

Therefore,
\[
L(A^{(1)})=\{\,w\in\{a,b\}^* : |w|\ge 1 \text{ and $w$ contains no substring $aa$ or $bb$}\,\}.
\]

Equivalently,
\[
L(A^{(1)}) = a(ba)^*(\varepsilon \mid b)\ \cup\ b(ab)^*(\varepsilon \mid a).
\]

\medskip
\underline{$A^{(2)}$:}

\begin{center}
\begin{tikzpicture}[shorten >=1pt,node distance=2.5cm,on grid,auto]
\node[state,initial] (1) {$1$};
\node[state,accepting] (2) [right=of 1] {$2$};
\node[state] (3) [below=of 1] {$3$};

\path[->]
(1) edge node {$a$} (2)
    edge node[left] {$b$} (3)
(2) edge[loop above] node {$a,b$} ()
(3) edge[loop below] node {$a,b$} ();
\end{tikzpicture}
\end{center}

The only accepting state is $2$. From state $1$, reading $b$ leads to a sink.
Reading $a$ moves $1 \to 2$, and from $2$ any symbol returns to $1$.

Thus a word must:
\begin{itemize}
\item start with $a$,
\item have odd length,
\item have $a$ in every odd position.
\end{itemize}

Hence,
\[
L(A^{(2)}) = a\big((a\mid b)a\big)^*.
\]

\medskip
\textbf{2. Intersection automaton $A = A^{(1)} \times A^{(2)}$.}

\begin{itemize}
\item States: $Q = Q_1 \times Q_2$
\item Start state: $(1,1)$
\item Accepting states:
\[
F = F_1 \times F_2 = \{2,4\}\times\{2\}
\]
\item Transition function:
\[
\delta((p,q),x) = (\delta_1(p,x), \delta_2(q,x))
\]
\end{itemize}

The reachable accepting state is $(2,2)$.

\medskip
\textbf{3. Proof that $L(A)=L(A^{(1)})\cap L(A^{(2)})$.}

For any word $w$, by induction on prefixes, after reading $w$ the product automaton is in state
\[
(\delta_1(1,w),\delta_2(1,w)).
\]
Thus $A$ accepts $w$ iff both component automata accept $w$.
Therefore,
\[
L(A)=L(A^{(1)})\cap L(A^{(2)}).
\]

\medskip
\textbf{4. Construction of union automaton $A'$.}

Keep the same product states and transitions, but define the accepting set as
\[
F' = (F_1\times Q_2)\ \cup\ (Q_1\times F_2).
\]
Then $A'$ accepts whenever at least one component accepts, so
\[
L(A') = L(A^{(1)}) \cup L(A^{(2)}).
\]

\medskip
\textbf{3. Proof that $L(B)=L(B^{(1)})\cap L(B^{(2)})$.}

After reading any word $w$, the product automaton is in state
\[
(\delta_1(p_0,w),\delta_2(q_0,w)).
\]
Thus $B$ accepts $w$ iff both component automata accept $w$.
Therefore,
\[
L(B)=L(B^{(1)})\cap L(B^{(2)}).
\]

\medskip
\textbf{4. Construction of union automaton $B'$.}

Using De Morgan's law:
\[
L(B^{(1)})\cup L(B^{(2)})
=
\overline{\ \overline{L(B^{(1)})}\ \cap\ \overline{L(B^{(2)})}\ }.
\]

\begin{enumerate}
\item Complement $B^{(1)}$ and $B^{(2)}$ by swapping accepting and non-accepting states.
\item Construct their product automaton for intersection.
\item Complement the resulting automaton.
\end{enumerate}

This yields $B'$ such that
\[
L(B') = L(B^{(1)}) \cup L(B^{(2)}).
\]


\paragraph{Exercise 2.2.7}
Let $A$ be a DFA and let $q$ be a state such that $\delta(q,a)=q$ for all input symbols $a\in\Sigma$.
We prove by induction on $|w|$ that for all strings $w\in\Sigma^*$, $\hat{\delta}(q,w)=q$.

\begin{proof}
\textbf{Base case:} $|w|=0$, so $w=\varepsilon$.
By definition of $\hat{\delta}$, $\hat{\delta}(q,\varepsilon)=q$.

\textbf{Inductive step:} Assume that for some $n\ge 0$, for every string $x\in\Sigma^*$ with $|x|=n$ we have $\hat{\delta}(q,x)=q$.
Let $w\in\Sigma^*$ be any string with $|w|=n+1$.
Then $w$ can be written as $w=xa$ where $|x|=n$ and $a\in\Sigma$ is the last symbol.
Using the recursive definition of $\hat{\delta}$,
\[
\hat{\delta}(q,xa)=\delta(\hat{\delta}(q,x),a).
\]
By the inductive hypothesis, $\hat{\delta}(q,x)=q$, so
\[
\hat{\delta}(q,xa)=\delta(q,a)=q,
\]
where the last equality uses the assumption that $\delta(q,a)=q$ for all $a\in\Sigma$.
Thus $\hat{\delta}(q,w)=q$ for all $w$ of length $n+1$.

By induction, $\hat{\delta}(q,w)=q$ for all $w\in\Sigma^*$.
\end{proof}

\paragraph{Exercise 1 -- Kleene's Algorithm}

Consider the DFA with states $p$ and $q$, where $p$ is the start and accepting state, and the transitions are:
\[
p \xrightarrow{a} p,\quad p \xrightarrow{b} q,\quad q \xrightarrow{a} p,\quad q \xrightarrow{b} q.
\]
Let $1=p$ and $2=q$.

For Kleene's algorithm, the base regular expressions are:
\[
R_{11}^{(0)}=\varepsilon + a,\qquad
R_{12}^{(0)}=b,\qquad
R_{21}^{(0)}=a,\qquad
R_{22}^{(0)}=\varepsilon + b.
\]

Using the recurrence
\[
R_{ij}^{(k)} = R_{ij}^{(k-1)} + R_{ik}^{(k-1)}\left(R_{kk}^{(k-1)}\right)^*R_{kj}^{(k-1)},
\]
we compute the next stage:

\[
R_{11}^{(1)} = a^*
\]
\[
R_{12}^{(1)} = a^*b
\]
\[
R_{21}^{(1)} = aa^*
\]
\[
R_{22}^{(1)} = \varepsilon + a^*b
\]

Then
\[
R_{11}^{(2)} = R_{11}^{(1)} + R_{12}^{(1)}\left(R_{22}^{(1)}\right)^*R_{21}^{(1)}.
\]

A simpler equivalent regular expression for the language is:
\[
R = \varepsilon + (a+b)^*a.
\]

Thus, the language accepted by the DFA is:
\[
L(A)=L(R)=\{\varepsilon\}\cup\{w\in\{a,b\}^* : w \text{ ends in } a\}.
\]

\paragraph{Exercise 2 -- Testing Equivalence of Regular Languages}

We apply the table-filling algorithm to the two DFAs in ITALC Exercise 4.4.2.

The first DFA has states $A,B$, where $A$ is accepting and $B$ is non-accepting.  
The second DFA has states $C,D,E$, where $C$ and $D$ are accepting and $E$ is non-accepting.

The accepting states are:
\[
A,\ C,\ D
\]
and the non-accepting states are:
\[
B,\ E.
\]

First, we mark all pairs where one state is accepting and the other is not:
\[
(A,B),\ (A,E),\ (C,B),\ (C,E),\ (D,B),\ (D,E).
\]

The remaining pairs are:
\[
(A,C),\ (A,D),\ (B,E),\ (C,D).
\]

Now we check transitions:

\begin{itemize}
    \item $(A,C)$:
    \[
    \delta(A,0)=A,\ \delta(C,0)=D \Rightarrow (A,D),
    \qquad
    \delta(A,1)=B,\ \delta(C,1)=E \Rightarrow (B,E)
    \]
    Neither resulting pair is marked, so $(A,C)$ remains unmarked.

    \item $(A,D)$:
    \[
    \delta(A,0)=A,\ \delta(D,0)=D \Rightarrow (A,D),
    \qquad
    \delta(A,1)=B,\ \delta(D,1)=E \Rightarrow (B,E)
    \]
    So $(A,D)$ remains unmarked.

    \item $(B,E)$:
    \[
    \delta(B,0)=A,\ \delta(E,0)=C \Rightarrow (A,C),
    \qquad
    \delta(B,1)=B,\ \delta(E,1)=E \Rightarrow (B,E)
    \]
    So $(B,E)$ remains unmarked.

    \item $(C,D)$:
    \[
    \delta(C,0)=D,\ \delta(D,0)=D,
    \qquad
    \delta(C,1)=E,\ \delta(D,1)=E
    \]
    So $(C,D)$ remains unmarked.
\end{itemize}

Since the start states $A$ and $C$ are not distinguishable, the two DFAs are equivalent.

Therefore,
\[
L_1 = L_2.
\]

In fact, both DFAs accept all binary strings that are either empty or end in $0$.
